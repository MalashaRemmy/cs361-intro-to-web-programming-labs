# Module 01 — Introduction to Internet Technologies

## Overview

This module introduces the foundational technologies, architecture, and communication systems that make the modern web possible.

The focus of this module is understanding how the Internet and the World Wide Web function at both a conceptual and practical level. Before building web applications, it is important to understand how browsers, servers, protocols, hosting systems, and networking standards interact to deliver web content across the Internet.

This module also introduces the workflow and tools used in modern web development environments.

---

## Learning Objectives

By the end of this module, I should be able to:

- Explain the basic architecture of the Internet and the World Wide Web
- Describe how HTTP and HTTPS communication works
- Understand the client-server model
- Explain the role of DNS, IP addresses, and web hosting
- Set up a complete local web development environment
- Use browser developer tools to inspect HTTP requests and responses
- Deploy a simple static web page online

## Key Takeaways

This module established the foundational understanding required for the rest of the course. Understanding how web systems communicate internally provides important context for frontend development, backend development, databases, APIs, deployment, and web security later in the course.

Rather than immediately focusing only on coding, this module emphasised understanding the environment and infrastructure that web applications depend on.
