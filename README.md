# CS361 — Introduction to Web Programming Labs

## Overview

This repository documents my learning journey through CS361 — Introduction to Web Programming at the Copperbelt University.

The repository serves as:
- a laboratory archive,
- a structured web development learning resource,
- a technical documentation portfolio,
- and a progressive exploration of modern web technologies from foundational HTML to full-stack web applications.

The course combines theoretical understanding with practical implementation across frontend, backend, databases, APIs, deployment, and web architecture concepts.

---

## Course Topics

- Internet Technologies
- HTML5 & Semantic Web Design
- CSS3 & Responsive Design
- JavaScript & DOM Manipulation
- XML, XPath & XSLT
- PHP Server-Side Development
- MySQL Database Integration
- REST APIs & Web Services
- Multi-Tier Architecture
- Web Security Fundamentals
- Git & GitHub Workflow

---

## Learning Philosophy

This repository is intentionally built progressively.

Each module focuses not only on implementation, but also on:
- understanding underlying concepts,
- documenting learning,
- improving engineering practices,
- and building maintainable, readable systems.

The goal is not merely to complete coursework, but to develop long-term competence in web engineering.
